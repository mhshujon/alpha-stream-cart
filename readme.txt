Transform your WooCommerce store with live streaming and real-time shopping!

== Description ==

StreamCart integrates your WooCommerce store with Facebook Live streaming, enabling real-time product showcasing and shopping experiences. Boost engagement and sales by connecting directly with your audience through live video.

Features:

* Facebook Live streaming integration
* Real-time product showcase
* Live comments display
* WooCommerce cart integration
* Basic stream statistics
* Easy-to-use admin interface

== Installation ==

1. Upload the plugin files to `/wp-content/plugins/streamcart`
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Configure your Facebook App credentials in the StreamCart settings
4. Start streaming and selling!

== Frequently Asked Questions ==

= Do I need a Facebook account? =

Yes, you need a Facebook account and a Facebook App with Live Stream permissions.

= Is WooCommerce required? =

Yes, StreamCart requires WooCommerce to be installed and activated.
